# 涨停复制——股票操作方法归纳

## 一、什么是“涨停复制”？

“涨停复制”是指一只股票在短期内出现过涨停板，经过一段时间的调整后，再次出现涨停板，形成第二波上涨。投资者通过捕捉这种模式，获取类似于第一波主升浪的收益。


## 二、涨停复制产生的原因

| 类别 | 具体因素 |
| :--- | :--- |
| **技术面** | 均线多头排列、MACD金叉、成交量上涨放量/下跌缩量 |
| **市场情绪** | 属于热点概念、题材炒作，受消息面（政策、事件）驱动 |
| **心理预期** | 投资者对强势股有继续上涨的预期，产生跟风买入意愿 |


## 三、具体操作方法

### 1. 选股条件

- 近期（1周内）曾出现**至少一个涨停板**，最好为连续涨停。
- 个股属于**当前市场热点题材**，有持续的消息刺激。
- 技术形态良好：均线呈多头排列，上涨放量、回调缩量。

### 2. 买入时机

- **不追高**：等待第一波涨停后的回调。
- **寻找支撑位**：如5日/10日均线、涨停板实体中点或开盘价。
- **缩量企稳**：回调时成交量明显缩小，表明主力未出逃。
- **再次放量启动**：股价在支撑位出现阳线，成交量重新放大时，尝试买入。

### 3. 仓位与风控（最重要）

- **控制仓位**：只用部分资金参与，避免全仓押注。
- **设置止损**：通常设在回调最低点下方或关键均线下方（如-5%~-8%、20日均线），跌破坚决卖出。
- **分散投资**：资金分配到不同板块或个股，降低单一风险。

### 4. 卖出与止盈

- 成功复制涨停或连续大涨后，当出现**放量滞涨**或**无法封住涨停**时，分批止盈。
- 跌破**5日均线**或**趋势线**时，全部卖出。
- 避免贪婪，不追求每次都复制多个涨停。

### 5. 如何获取热点题材

- 近10个交易日中，累计涨停家数最多的5个板块即可视为热点板块
